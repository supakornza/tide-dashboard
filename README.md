# 🌊 Tide Monitor Dashboard — คู่มือการติดตั้ง

## ระบบติดตามระดับน้ำขึ้น-น้ำลง
### โครงการป้องกันและแก้ไขปัญหาการกัดเซาะชายฝั่งทะเลบางขุนเทียน

---

## 📋 สิ่งที่ต้องเตรียม

1. **บัญชี Google** (Gmail) — ใช้สำหรับ Google Apps Script
2. **Google Maps API Key** — สำหรับแสดงแผนที่ (ฟรี เดือนละ ~200 ครั้ง)
3. **เว็บเบราว์เซอร์** — Chrome, Edge, Firefox

---

## 🚀 วิธีติดตั้ง (ทำครั้งเดียว)

### ขั้นตอนที่ 1: สร้าง Google Apps Script

1. เปิดเบราว์เซอร์ไปที่ **https://script.google.com**
2. คลิก **"New Project"** (โปรเจกต์ใหม่)
3. ตั้งชื่อโปรเจกต์ว่า: `Tide Monitor Dashboard`

### ขั้นตอนที่ 2: สร้างไฟล์

ในโปรเจกต์ ให้สร้างไฟล์ 2 ไฟล์:

**ไฟล์ที่ 1: `Code.gs`**
- ลบโค้ดเดิมออกทั้งหมด
- วางโค้ดจากไฟล์ `Code.gs` ที่แนบมาทั้งหมด
- กด **Ctrl+S** บันทึก

**ไฟล์ที่ 2: `index.html`**
- คลิก **"+"** ข้าง ๆ ชื่อไฟล์ > เลือก **"HTML"**
- ตั้งชื่อว่า `index`
- ลบโค้ดเดิมออกทั้งหมด
- วางโค้ดจากไฟล์ `index.html` ที่แนบมาทั้งหมด
- กด **Ctrl+S** บันทึก

### ขั้นตอนที่ 3: ใส่ Google Maps API Key (ถ้าต้องการแผนที่)

1. ไปที่ **https://console.cloud.google.com**
2. สร้างโปรเจกต์ใหม่ (หรือเลือกที่มีอยู่)
3. เปิดใช้งาน **"Maps JavaScript API"**
4. สร้าง **API Key** ที่ **Credentials**
5. กลับไปที่ Google Apps Script
6. ในไฟล์ `index.html` ค้นหา `YOUR_GOOGLE_MAPS_API_KEY`
7. แทนที่ด้วย API Key ของคุณ

> 💡 **ถ้าไม่มี API Key** แผนที่จะไม่แสดง แต่ส่วนอื่นยังใช้งานได้

### ขั้นตอนที่ 4: Deploy เป็น Web App

1. คลิก **"Deploy"** (ปุ่มมุมขวาบน) > **"New deployment"**
2. คลิก **"Settings"** (เฟือง) > เลือก **"Web app"**
3. ตั้งค่า:
   - **Description**: `Tide Monitor v1.0`
   - **Execute as**: `Me`
   - **Who has access**: `Anyone` (หรือ `Anyone with Google account` ถ้าต้องการจำกัด)
4. คลิก **"Deploy"**
5. กด **"Authorize access"** และยอมรับสิทธิ์
6. คัดลอก **Web App URL** ที่ได้รับ

### ขั้นตอนที่ 5: เปิดใช้งาน

1. วาง URL ที่ได้ลงในเบราว์เซอร์
2. Dashboard จะแสดงขึ้นมาพร้อม:
   - 🌊 ระดับน้ำปัจจุบันของทั้ง 3 สถานี
   - 📈 กราฟน้ำขึ้นน้ำลงตลอดวัน
   - 🗺️ แผนที่พื้นที่โครงการ
   - 📅 พยากรณ์ 7 วัน
   - 🏗️ ช่วงเวลาแนะนำเข้างาน

## 📊 คำอธิบายเมนู

### แถบควบคุม
- **📅 วันที่**: เลือกวันที่ต้องการดูข้อมูล
- **สถานี**: เลือกสถานีอ้างอิง (เปลี่ยนกราฟและตาราง)
- **วันนี้**: กลับไปวันปัจจุบัน
- **🔄 รีเฟรช**: โหลดข้อมูลใหม่

### การ์ดระดับน้ำ
- แสดงระดับน้ำปัจจุบันของทั้ง 3 สถานี
- สีฟ้า = ปกติ, สีส้ม = น้ำสูง, สีเขียว = น้ำต่ำ
- คลิกเพื่อเลือกสถานี

### กราฟ
- เส้นฟ้า = ระดับน้ำตลอดวัน
- เส้นเหลืองประ = ระดับน้ำทะเลเฉลี่ย (MSL)
- พื้นที่เขียว = ช่วงเวลาเข้างานได้

### ช่วงเวลาเข้างาน
- แสดงช่วงเวลาที่ระดับน้ำเหมาะสม
- เหมาะสำหรับเคลื่อนย้ายเครื่องจักร ขนส่งวัสดุ
